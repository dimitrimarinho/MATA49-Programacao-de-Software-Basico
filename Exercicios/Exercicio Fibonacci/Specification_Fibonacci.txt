Implementar uma solução recursiva para o cálculo do número de fibonacci

F(0) = 0
F(1) = 1
F(2) = F(1) + F(0) = 1
F(3) = F(2) + F(1) = 2
F(4) = F(3) + F(2) = 3
F(5) = F(4) + F(3) = 5
F(6) = F(5) + F(4) = 8
...
F(n) = F(n-1) + F(n-2)

Exemplo:
        Entrada = 4
        Saída = 3

        Entrada = 6
        Saída = 8
